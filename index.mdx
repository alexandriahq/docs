Ambient is memory for your computer.

Install the desktop app, choose what Ambient can capture, then keep working. Ambient builds a private memory of your active work and prepares handoffs for tools like Codex and Claude Code.

## How it works

Ambient stays out of the way by default.

<Steps>
  <Step title="Install">
    Download the latest Ambient app for macOS or Windows.
  </Step>

  <Step title="Choose what to capture">
    Pick the apps, windows, and activity sources Ambient can use.
  </Step>

  <Step title="Keep working">
    Ambient runs in the background and builds memory from your normal workflow.
  </Step>

  <Step title="Hand off work">
    When Ambient sees a task an agent can help with, it prepares the context and notifies you.
  </Step>
</Steps>
